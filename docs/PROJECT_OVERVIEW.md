## Project Description

AI-X is an educational web game designed to teach Artificial Intelligence concepts through interactive gameplay, quizzes, and adaptive learning mechanics.
The project combines a lightweight custom game engine with educational systems to create an engaging and scalable learning experience inside the browser.

## Core Vision

The main vision of AI-X is to make AI education interactive, accessible, and creative instead of purely theoretical.
Players learn by interacting with lessons, quizzes, and challenges embedded inside the game world.

## Target Audience

* Students learning Artificial Intelligence fundamentals
* Beginner programmers
* Educational technology enthusiasts
* Portfolio and research audiences

## Platform

* Web Browser (Primary)
* Desktop-first experience
* Offline-compatible architecture

## Development Status

In active development.
Core engine, menu system, account system, and scene architecture are implemented.
Educational systems and content expansion are in progress.

## Unique Value

Unlike traditional educational platforms, AI-X integrates:

* Game mechanics
* Narrative learning
* Adaptive AI assistance
* Interactive quizzes
* Account-based progress tracking
