# AI-X

AI-X is a lightweight AI-driven game engine focused on modular systems, logic automation, and intelligent behaviors.

## Features
- Modular architecture
- AI decision systems
- Event-driven engine
- Lightweight and fast
- Browser-based runtime

## Getting Started
1. Clone the repository
2. Open public/index.html
3. Run the engine in the browser

## Project Structure
engine/
core engine logic

systems/
ai and gameplay systems

public/
game runtime and entry point
